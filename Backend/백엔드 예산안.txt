1. 전제 조건

- 실제 서비스 배포 (외부 접속 가능)
- OAuth 로그인 정상 동작
- DB 포함
- HTTPS 적용
- 시연 안정성 확보 (다운되면 안됨)
- 개발 단계에서는 로컬 환경을 사용하고,  
  최종 검증 및 시연 단계에서 클라우드 인프라를 적용한다

---

2. 인프라 구성

- 서버: Amazon Web Services EC2
- DB: 서버 내부 (MySQL / PostgreSQL)
- 배포: Docker
- 도메인 + HTTPS 적용
- OAuth: Google
